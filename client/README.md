# AdClip AI 🎬

AI-powered advertisement creative generator that turns product images into professional advertisement photos and videos for social media and commercials.

## 🚀 Live Demo

(https://ad-clip-ai.vercel.app/)

## ✨ Features

- 🖼️ AI Image Generation (Flux by Pixazo)
- 📝 AI Script Generation (Gemini)
- 🎬 Video Generation (Coming Soon)
- 👥 Community Gallery
- 💳 Credits System
- 🔐 Auth (Clerk)

## 🛠️ Tech Stack

_Frontend:_ React, TypeScript, Vite, TailwindCSS  
_Backend:_ Node.js, Express, TypeScript  
_Database:_ PostgreSQL (Neon) + Prisma  
_AI:_ Gemini API, Pixazo Flux API  
_Storage:_ Cloudinary  
_Auth:_ Clerk  
_Deployment:_ Vercel + Render

## 🔧 Setup

```bash
# Install dependencies
cd client && npm install
cd server && npm install

# Add environment variables
# client/.env and server/.env

# Run
npm run dev
```
